# VCV Rack Tutorial

A hands-on tutorial for learning VCV Rack -- the open-source Eurorack modular synthesizer simulator.

## What is VCV Rack?

VCV Rack is a free, open-source virtual modular synthesizer that emulates the Eurorack hardware format. You build sounds by patching virtual cables between modules: oscillators, filters, envelopes, sequencers, effects, and more. It runs on Windows, Mac, and Linux.

- Download: https://vcvrack.com/
- Manual: https://vcvrack.com/manual/

## Tutorial Outline

### Part 1: Getting Started
- Installing VCV Rack
- The interface: modules, cables, the rack
- Your first patch: oscillator -> filter -> VCA -> output

### Part 2: Core Concepts
- Voltage and signals: audio vs. CV vs. gate
- Oscillators (VCO): pitch, waveforms
- Filters (VCF): cutoff, resonance, filter types
- Amplifiers (VCA): controlling volume with CV
- Envelopes (ADSR): shaping a sound over time

### Part 3: Modulation
- LFOs: slow oscillators as modulation sources
- Patch routing: modulating filter cutoff, pitch, etc.
- Random and sample-and-hold

### Part 4: Sequencing
- Step sequencers
- Clock modules
- Building a basic rhythm patch

### Part 5: Polyphony and Advanced Topics
- VCV Rack polyphony system
- Effects: reverb, delay, chorus
- Using the VCV Library to find modules
- Saving and sharing patches

## Notes
